NodeSchool Workshopin luominen

Tässä muutamia yleisesti käytettyjä työkaluja NodeSchool workshopin luomiseen.

workshopper

npm install workshopper täysimittainen nodeschool CLI workshop framework.

adventure

npm install adventure kevyt vaihtoehto workshopperille.

Luo omasi

Älä pelkää formaatin hakkerointia! Jos esimerkiksi haluat kirjoittaa workshopin joka opettaa CSS:ää, voit joutua keksimään oman workshop formaatin. Esimerkiksi browser-menu moduuli on kirjoitettu shader-school workshopille. browser-menu module was written for the shader-school workshop.
