    Rozpocznij —

    Potrzebujesz Node.js na Twoim komputerze, żeby zacząć swoją przygodę z zadaniami. Następnie skorzystaj z npm (instalowany razem z Node), żeby zainstalować każdy z modułów za pomocą podanych poleceń. Po pomyślnej instalacji, wpisz nazwę wybranego Workshoppera, żeby go uruchomić.

    Będziesz również potrzebować edytora tekstowego, jeśli jeszcze go nie masz, polecamy te przeznaczone dla programistów: Atom, Sublime Text, Visual Studio Code, Brackets.

    Problems with the workshopper? Try installing the latest version of Node.js (v6.2.0 or higher), or ask a question in the discussion.
